TDA 기반 상품 식별

Persistent Homology를 활용한 PH 바코드 실험

물류센터에서 바코드가 훼손되거나 누락된 경우, 상품 포장 정면을 스마트폰으로 촬영하여 Persistent Homology 기반의 PH 바코드를 생성하고, 이를 통해 상품을 식별할 수 있는지 실험적으로 검증한다.

학습 데이터 불필요, GPU 없이 CPU만으로 계산 가능 — 중소 창고에서도 도입할 수 있는 경량 시스템이 목표.

Why — 현업 배경

물류센터에서 상품 구분은 거의 전적으로 바코드에 의존한다. 바코드가 훼손되면 작업자가 육안으로 확인해야 하고, 오분류와 속도 저하가 발생한다. RFID는 태그 단가, 비전처리는 GPU 비용 문제로 중소 창고에서는 도입이 어렵다. TDA는 학습 데이터 없이 사진 한 장에서 바로 특징을 추출할 수 있다.

Setup Guide
1 GitHub 레포 생성
portfolio — public 정적 HTML 포트폴리오. GitHub Pages 배포 가능.
tda-api — private Spring Boot + Python FastAPI 모노레포 (backend/ + tda-engine/)
tda-android — private Android 촬영 앱
2 AWS EC2 생성
EC2 인스턴스 생성 — t3.small, Ubuntu 24.04 LTS 리전: ap-northeast-2 (서울)
보안그룹 설정 22 (SSH), 80 (HTTP), 443 (HTTPS) 인바운드 허용
Elastic IP 할당 & 연결 인스턴스 재시작해도 IP 유지
키 페어 생성 & SSH 접속 확인 ssh -i key.pem ubuntu@[EC2-IP]
3 AWS 비용 관리
Billing 알림 활성화 — AWS Console → Billing → Billing Preferences "Receive Free Tier Usage Alerts" + "Receive Billing Alerts" 체크
Budget 생성 — AWS Console → Billing → Budgets → Create Budget Monthly cost budget: $30 (약 4만원). 50%, 80%, 100% 도달 시 이메일 알림.
CloudWatch 비용 알람 → EC2 자동 중지 EstimatedCharges > $70 (약 10만원) 시 SNS 알림 + Lambda로 EC2 stop 자동 실행
간단 대안: AWS Instance Scheduler 또는 cron 기반 야간 중지 새벽 2시~8시 자동 중지로 월 ~25% 절감 가능. 실험 단계에서는 선택사항.
4 서버 기본 셋팅
시스템 업데이트 sudo apt update && sudo apt upgrade -y
nginx 설치 & 기본 설정 포트폴리오 정적 파일 서빙 + Spring Boot 리버스 프록시
Java 17 (OpenJDK) 설치 sudo apt install openjdk-17-jdk -y
Python 3 + pip + venv 설치 sudo apt install python3 python3-pip python3-venv -y
MySQL 설치 & DB 생성 sudo apt install mysql-server -y
Git 설치 & GitHub SSH 키 등록 private 레포 clone을 위해 필요
5 도메인 & SSL
Cloudflare DNS — kyuwan.kim → EC2 Elastic IP A 레코드 추가, Proxy 꺼두기 (DNS only)
Let's Encrypt SSL 인증서 발급 sudo apt install certbot python3-certbot-nginx -y
nginx HTTPS 설정 & 자동 갱신 확인
6 포트폴리오 배포
portfolio 레포 clone → nginx 서빙 디렉토리에 연결 sudo ln -s /home/ubuntu/portfolio /var/www/portfolio
kyuwan.kim 접속 시 포트폴리오 페이지 표시 확인
7 백엔드 배포
tda-api 레포 clone backend/ (Spring Boot) + tda-engine/ (FastAPI)
Python 가상환경 생성 & 패키지 설치 python3 -m venv venv && pip install fastapi uvicorn gudhi opencv-python numpy
Spring Boot 빌드 & 실행 ./gradlew build && java -jar build/libs/app.jar
FastAPI 실행 uvicorn main:app --host 127.0.0.1 --port 8000
systemd 서비스 등록 — 재부팅 시 자동 시작 spring-boot.service + fastapi.service
nginx 리버스 프록시 — api.kyuwan.kim → Spring Boot :8080
