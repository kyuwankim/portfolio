TDA 기반 상품 식별

사진 한 장으로 상품의 "지문"을 만들 수 있을까?

물류센터에서 바코드가 손상되거나 떨어져 나가면,
작업자는 포장 외관을 보고 직접 상품을 판단해야 한다.

기존 대안에는 한계가 있다.
RFID는 태그 단가 문제로 중소 창고에 도입이 어렵다.
딥러닝(CNN)은 상품별 수백 장의 학습 데이터와 GPU가 필요하고,
신규 상품마다 재학습해야 한다.

접근 — Topological Data Analysis

이 연구는 TDA(위상 데이터 분석)를 활용해
상품 포장 이미지에서 고유한 수학적 패턴을 추출한다.

TDA는 데이터의 형태(topology)를 다양한 스케일에서 분석하는 수학적 프레임워크다.
노이즈에 강하고, 회전·스케일 변화에 안정적이다.

구체적으로 Persistent Homology를 사용한다.
이미지를 grayscale로 변환한 뒤 밝기 threshold를 0→255로 올리면서
위상적 구조의 변화를 추적한다.

H₀ — 연결 성분
밝은 영역이 나타나고(birth)
합쳐지는(death) 과정.
포장의 전체 밝기 분포를 반영.
H₁ — 루프
글자·로고·패턴의 닫힌 고리가
생성되고 소멸하는 과정.
디자인의 구조적 특징을 포착.

각 구조의 (birth, death) 쌍을 모으면 Persistence Diagram이 만들어진다.
이것이 해당 상품의 위상적 지문이다.
오래 살아남는(high persistence) 구조일수록 실제 특징일 가능성이 높다.

촬영 → Grayscale → Filtration → PH Diagram → 특징 벡터 → 거리 비교
비교 — 특징 벡터 & 거리 측정

Persistence Diagram에서 18차원 특징 벡터를 추출한다.

H₀/H₁ 각각의 count, total/max/mean/std persistence,
4개 구간의 Betti number curve 값으로 구성된다.

두 상품의 벡터 사이 L2 거리를 계산하여
임계값 이하이면 같은 상품으로 판별한다.

학습 데이터 불필요 CPU 실시간 계산 신규 상품 즉시 비교
가설 & 검증

같은 상품의 포장은 촬영 각도·거리·조명이 달라지더라도
유사한 위상적 특징 벡터를 생성할 것이며,
서로 다른 상품과는 유의미한 L2 거리 차이를 보일 것이다.

검증: 동일 상품을 다양한 조건에서 N장 촬영하여
intra-class(같은 상품 간) / inter-class(다른 상품 간) 거리를 비교한다.
두 분포가 통계적으로 분리되면 가설을 지지하는 것으로 판단한다.
